**3. Fuzzy regression model**

**3.1. Fuzzy regression analysis**

Fuzzy regression methods have considerably
widened the field of application of classical regression
methods in such a way that they are helpful in
determining regression relations from fuzzy initial data,
which can be either quantitative or qualitative.
Therefore, fuzzy regression methods can be applied to
expert estimation and several fields of human activities.
When applying regression analysis to predict random
variables based on the existing observed values, we
can see that the data obtained will be affected more or

less by random factors. Suppose that n sample points
have been gained, so if all data points approximately
reflect the actual things, the result gained will be
relatively reasonable. Such a result is very realistic but
seldom appears in practice. So when one or some data
points are greatly influenced by random factors, there
must be a difference between the result gained and the
actual values. At that time, the classical regression
analysis is not applicable.
In this paper, fuzzy least squares method is put
forward based on fuzzy expectation and fuzzy variance.
By fuzzy least squares method, the regression
parameters are estimated and the corresponding
regression model is established. Such is taken as fuzzy
regression analysis. Fuzzy regression analysis can
make the disturbance of morbid data and bad

information to the decision result decrease to the

minimum and raise accuracy and reliability of the
decision result accordingly.

perspective. The COSMIC-FFP method can measure
the functional size of almost all the software types. In
the early stage of software development, we can
measure the functional size from the specifications and
then estimate the effort to be spent based on the fuzzy
decision-making model. But in practice, some other
factors may influence the effort, such as project
difficulty, program language and LOC etc. So in the
late stage of software development (e.g. maintenance

according to the expression (7) and build up a new
