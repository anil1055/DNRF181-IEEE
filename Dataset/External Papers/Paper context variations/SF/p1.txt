**Abstract**

The advent of Internet of Things (IoT) has revolutionized the way information and services are delivered via the web.
The widespread popularity of web applications has seen a meteoric rise, with organizations of all sizes, both public and
private, opting to make their services and operations available online. However, the development of these web-based
applications has become increasingly complex, posing a challenge for organizations to manage this growing complexity
within budget and timeline constraints. To address this challenge, there is a pressing need for accurate and efficient web
effort estimation methods. The accuracy of such estimation has a direct and profound impact on the survival and success of
development companies. This research paper proposes a novel approach to web size estimation, the Web Use Case Points
(WCP) method, which takes into consideration functional and non-functional size to predict the size of a web application.
The study also compares the WCP method with the widely used Function Point (FP) and Web Object (WO) methods. The
results show that the size obtained through the WCP method is smaller than that obtained through the WO method, and
greater than that obtained through the FP method in almost all ten web projects analyzed in this study. In conclusion, the
WCP method provides a new and promising approach to web effort estimation, offering greater accuracy and efficiency
for organizations looking to navigate the complex landscape of web application development.

**Keywords** Web effort estimation · Web size metrics · Technical and environmental web factors · Web case points

**Web Case Points**

Web case points (WCP), a web size metrics proposed in this
study are based on functional, dimensional and nontechni-­
cal parameters required for web application development.
WCPs were obtained from requirements elicited and elabo-­
rated through objective oriented model-driven approach
called as use case modelling. In use case modelling, use
case diagrams are used to express requirements into their
corresponding functional and dimensional functional mea-­
sures. Actors, use-cases, use case scenarios and different
type’s relations and associations are main constructs in
objective oriented approach to model working framework
of web application development. The overall size of the web
application development is carried out in multiple steps:

**Results and Discussions**

The proposed web size metric, WCP, was rigorously evalu-­
ated in this study by comparing its estimated size with two
widely used metrics, FP and WO. Both FP and WO cal-­
culate the functional size of a web application. FP, a com-­
monly used sizing metric for software effort estimation, and
WO, the first web-specific size metric, were implemented
for effort estimation as described in references [11] and [9].

A dataset of 10 industrial web projects was employed to
assess the accuracy of FP, WO, and WCP in calculating web
application size. The results of the implementation are pre-­
sented in Table 4.

The results indicate that the size estimates obtained by
FP were smaller than both WO and WCP in the majority of
the 10 web projects. WO, on the other hand, produced larger
size estimates than FP and WCP. The proposed WCP met-­
ric was found to provide a balanced size estimate that falls
between the estimates obtained by FP and WO.

This study highlights the limitations of FP in web effort
estimation and underscores the need for a more comprehen-­
sive metric that takes into account all aspects and parameters
relevant to web application development. While WO was
specifically designed for web effort estimation, the results
show that WCP provides more balanced size estimates in
most of the web projects studied.

In conclusion, this study demonstrates the potential of
WCP to provide better size estimates for web application
development compared to commonly used FP and WO met-­
rics. The balanced size estimate obtained through WCP can
lead to more accurate effort estimates, making it a valuable
addition to the field of web effort estimation.
